# TAI - Trust & Authenticity Intelligence

TAI is a comprehensive multi-task learning platform designed to detect AI-generated content, assess truthfulness, and identify bias in text. It combines a modern Next.js frontend with a robust Python/RoBERTa backend.

## 🚀 Overview

The project aims to provide a "Trust Guard" for digital content, utilizing a fine-tuned RoBERTa-base model to perform three primary tasks:
1.  **Truthfulness Detection**: Estimating the veracity of a given text.
2.  **AI Origin Detection**: Identifying whether content was generated by a human or an AI.
3.  **Bias Analysis**: Categorizing content as No Bias, Left-leaning, or Right-leaning.

## 🛠️ Tech Stack

### Frontend
- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/)
- **State Management**: React Hooks & Context

### Backend
- **Framework**: [FastAPI](https://fastapi.tiangolo.com/) / [Flask](https://flask.palletsprojects.com/)
- **Language**: [Python 3.10+](https://www.python.org/)
- **Machine Learning**: [PyTorch](https://pytorch.org/), [Hugging Face Transformers](https://huggingface.co/docs/transformers/index)
- **Model**: [RoBERTa-base](https://huggingface.co/roberta-base) (125M Parameters)
- **Dataset**: Custom-curated 75,000+ sample multi-task dataset.

## 🔧 Installation & Setup

### Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the backend server:
   ```bash
   python main.py
   ```

### Frontend Setup
1. Install dependencies from the root directory:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

## 🧠 Model Training
The model is fine-tuned on a 75,000+ sample dataset using a multi-head architecture.
- **Base Model**: `roberta-base`
- **Epochs**: 2 (Recommended for fine-tuning)
- **Batch Size**: 16
- **Tasks**: Truth (MSE), AI (CE), Bias (CE)

Training progress can be monitored using the scripts in `backend/training/`.
